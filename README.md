Power BI Orders Analysis Report

This repository contains an interactive Power BI report analyzing orders, revenue, and profit by product category over time. It demonstrates an end-to-end analytics workflow, from data connection to interactive dashboard creation.

Objectives

Connect & Profile Data

Imported source data files

Conducted basic profiling and QA tasks

Familiarized with tables and fields

Create Relational Model

Defined relationships between fact and dimension tables

Created hierarchies

Optimized model properties for analysis

Add Calculated Measures & Fields

Enhanced the data model with new measures and calculated fields

Prepared the model for deeper analysis

Build Interactive Report

Developed an interactive dashboard to visualize orders, revenue, and profit over time and by product category

Added filters and slicers for dynamic exploration

Highlighted key metrics and trends for actionable insights

Key Features

Interactive filters and slicers

Trend analysis of revenue and profit over time

Breakdown of metrics by product category

Custom calculated measures for KPIs (e.g., total revenue, profit margin, order count)
